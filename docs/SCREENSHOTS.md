# 截图归档索引

## 说明

本文件用于记录项目截图素材的归档位置，避免根目录继续堆放零散图片。

## 归档规则

1. 与具体需求文档直接相关的截图，放到对应 `docs` 分类目录下的 `assets` 子目录。
2. 偏设计参考性质的截图，归入对应 PRD 的参考图目录。
3. 偏验收或回归验证性质的截图，归入 `docs/RELEASE/assets`。
